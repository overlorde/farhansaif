Papers

State of Security in WebAssembly Binaries

  • Summary: Finding vulnerabilities in binaries and exploiting them is a well researched field in computer security. With the advent of new byte code format for web - WebAssembly, classic techniques as well as new and innovative exploitation possibilities are arising as a new field of study. In this paper, we discuss about the prospects of vulnerabilities in WebAssembly binaries and their possible mitigation as proposed in existing studies.
