About My Research Interest

My research intrest includes systems programming and vulnerability analysis for WebAssembly binaries.

Also another thing that fascinates me is the development of different kinds of compilers, implemented some Just-In-Time compilation processes and saw how the code converts from source to machine code with optimizations on. Currently working on two projects - one is to use code property graph to make a vulnerability detector for WebAssembly binaries and other one is to learn more about different compilation processes and how they work, how their code optimizations make binaries faster.

My Recent Papers

State of Security in WebAssembly Binaries

Finding vulnerabilities in binaries and exploiting them is a well researched field in computer security. With the advent of new byte code format for web - WebAssembly, classic techniques as well as new and innovative exploitation possibilities are arising as a new field of study. In this paper, we discuss about the prospects of vulnerabilities in WebAssembly binaries and their possible mitigation as proposed in existing studies.
